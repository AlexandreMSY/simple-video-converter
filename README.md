# Simple Video Converter


Simple video converter is a Java program for dealing with video conversion.

## Instalation 
1 - Clone the repository 

```sh
git clone git@github.com:AlexandreMSY/simple-video-converter.git
```

2 - Open the folder and browse to 
```sh
/src/main/com.example.videoproject.
```

3 - Execute the App.java class.

## Usage

Click Add Video and select the video that you want to convert, then choose an output format and click Convert.
